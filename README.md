# ⌨️ Key Detector

A simple **Key Detector** web application built using HTML, CSS, and JavaScript.  
It detects and displays the **key pressed by the user** along with its key code in real time.

---

## 🚀 Features
- Detects keyboard key presses
- Displays pressed key instantly
- Shows key and key code
- Simple and responsive UI
- Beginner-friendly JavaScript logic

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)


---

## ▶️ How It Works
1. Open the application
2. Press any key on your keyboard
3. The pressed key and its code will be displayed on the screen
